## 개요

**연운 (Where Winds Meet)** 장비 강도 비교·최적화 도구입니다.
<연운> 공식-데이터 툴에서 가져온 캐릭터 스탯과、 연운 데미지 공식을 기반으로 **무격 지수** 를 산출합니다.
최적 장비 제안、 장비 개별 비교 등 장비 개선 방향을 제시합니다.

---

## 주요 기능

### 📥 데이터 가져오기
- <연운> 공식-데이터 툴에서 **원클릭으로 캐릭터 정보 가져오기** → 조율·정음 등을 포함한 장비 정보 / 장착 중인 심법 등 캐릭터 정보 를 일괄 반영.
- 수동 입력 없이 무격 지수 즉시 표시.

### 🏯 무격 지수
- 캐릭터 스탯을 기반으로、 모든 플레이어 공통의 고정 스킬 계수로 데미지 기댓값을 산출한 지표.
- 무술 스킬 배율 차이 등을 고정화했기 때문에 장비의 절대 강도를 비교할 수 있습니다.

### 🪶 장비 비교
- 가져온 장비와 새 장비 간의 무격 지수 변동을 실시간 프리뷰.
- 새 장비에 대해 편리한 OCR 가져오기 기능 (전 언어 대응) 탑재.

### 🧘 심법 비교
- Tier 별 효과 리스트 + 무술 속성 전용 효과 반영.
- 현 단계에서는 T2/T5 이외의 효과를 정확히 점수화하기 어렵기 때문에、 글로벌 데미지 계수를 곱합니다.

### 📊 격석 패널
- **기댓값 랭킹**: 현재 장비를 기반으로 조율/정음의 기댓값 랭킹을 볼 수 있습니다.
- **장비 최적화**: 현재 장비로부터의 최적 장비 제안/적용 기능.

### 📈 무격 이력
- 과거 가져오기와 무격 지수 추이를 그래프로 표시.
- 다중 캐릭터 대응、 기간 전환 (전 기간 / 30일 / 7일).

### 💾 프리셋 저장
- 멀티 슬롯 저장 / 읽기 / 삭제.
- 시행 중인 장비 구성 + 기준값 저장.

### 📡 OBS Share
- 색상 커스터마이즈 가능한 스탯 정보를 OBS 로 송출할 수 있는 오버레이 URL 발행.

### 🌐 다국어 대응 UI
- 日本語 / English / 简体中文 / 한국어 / Tiếng Việt — 전 UI·데이터 대응.
- ※ 표기가 이상한 부분 등 있으면 연락 부탁드립니다.

## 무격 지수 Tier 판정 기준

가져오기 시 적용된 장비로 산출된 무격 지수와 최적화 장비 기준의 무격 지수의 비율로 판정합니다.
무격 지수 Tier 는 가져오기 시 자동 판정되므로 다시 가져올 때까지 변동하지 않습니다.

| Tier | 비율 |
|---|---|
| **SS** | 최대의 95% 이상 |
| **S** | 90% 이상 |
| **A** | 80% 이상 |
| **B** | 65% 이상 |
| **C** | 미만 |

## 계산에 반영되는 효과

- 장비 기본 스탯 (외공 공격력 / 무술 속성별 공격력 등)
- 조율 / 정음 효과
- 무술 천부 효과 (회심률 상한 +Δ、 무술 속성별 공격 / 관통 / 속성 데미지 강화 등)
- 심법 Tier 효과 (T2/T5 = 표시 반영、 T0/T1/T3/T4/T6 = 숨겨진 가산)
- 2세트 효과 (2개 장비로 발동하는 가산계)
- 4세트 효과 (4개 장비로 +100 고정 보너스、 각 장비에 균등 배분)
- 기본 스탯 (체/력/방/속/회) → 파생 (기본 스탯·판정 확률)
- 캐릭터 기본 스탯 (천부 / 오음지태평 / 첩음 등)

---

## 계산에 반영되지 않는 효과

- PvP 전용 정음 (정음 6 슬롯) — 표시만、 계산 기여 없음

---

## 게임 내 스탯과의 오차에 대해

무격 지수는 캐릭터 기본 스탯 (천부 / 오음지태평 / 첩음 등) 을 **모든 플레이어 공통의 최대치** 로 가정하여 산출합니다.
캐릭터가 아직 최대치에 도달하지 않은 경우、 도구의 수치가 실제 게임 내 수치보다 높게 나타날 수 있습니다.
이는 장비의 절대 강도 비교를 성립시키기 위한 사양입니다.
