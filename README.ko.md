## 개요

**풍연전 (Where Winds Meet)** 장비 강도·최적화 분석 도구입니다.
게임 공식 데이터 도구 (확장) 에서 임포트한 캐릭터 데이터를 기반으로、 **무격지수 (Martial Index)** 를 중심으로 장비·조율/정음 옵션·무술·심법·장비 한 식 효과를 통합하여 데미지 기댓값을 산출합니다.

> v2.0.0 에서 구버전의 수동 스테이터스 입력 UI 를 폐지하고、 장비 데이터 구동의 신 UI 로 전면 이행했습니다.

## 주요 기능

### 📥 데이터 임포트 (완전 자동)
- 게임 공식 데이터 도구 (확장) 에서 **원클릭으로 캐릭터 데이터 임포트** → 장비 10개 / 조율·정음 옵션 / 심법 (4슬롯 + 무고) / 무술 / 캐릭터 정보 를 일괄 반영.
- 수동 입력 없이 무격지수 즉시 표시.

### 🏯 무격지수 (Martial Index)
- 장비·조율/정음·무술·심법·장비 한 식 효과 를 모두 포함하여、 전 플레이어 공통의 **고정 스킬 계수** 로 데미지 기댓값을 산출한 지표.
- 대세계 등급과 적측 파라미터의 개인차를 배제 → 장비의 절대 강도를 비교 가능.

### ⚔️ 장비 카드 / 심법 카드
- 장비 카드: 무술 속성별 (강명 / 쇄암 / 사조 / 순람 / 무상) 색 구분 + 장비 한 식 효과명 + 무술명 + 점수.
- 심법 카드: Tier (T1〜T5) 인장 chip + 심법명 + 점수.

### 🪶 장비 비교 modal (두루마리 UI)
- 신규 (가상) 장비로 인한 스테이터스 변동 → Δ Score 즉시 프리뷰.
- MAX% 등급별 색상 (금 / 보라 / 파랑)、 **100% 도달 시 무지개 색조 회전 애니메이션**.
- PvP 전용 정음 (정음 6) 대응、 신규 장비 레벨 상한은 캐릭터 레벨로 제한.

### 🧘 심법 modal
- 장비 비교와 통일 UI、 Tier 효과 리스트 (T0〜T6) + 무술 속성 전용 effects 반영.

### 📊 격석 패널 (탭식)
- **기댓값 랭킹**: 조율/정음 옵션 +N 의 Δ Score 일람.
- **스테이터스 효율 분석**: 스테이터스 +1 당 ΔScore.
- **장비 최적화**: 전 장비의 Greedy 전체 최적화 제안.
- **이력**: 과거 임포트 / Δ 일람.

### 💾 프리셋 저장
- 멀티 슬롯 저장 / 읽기 / 삭제.
- importSnap + state + 가상 장비 + 기준값 (baseline) 동봉.

### 📡 OBS Share / 송출용 URL
- 표시 전용 URL (`?view=sidebar`) 로 sidebar 만 OBS overlay 화.
- 색상 커스터마이즈 (배경 / 불투명도 / 문자색 등 6항목).

### 🌐 다국어 (4언어)
- 日本語 / English / 简体中文 / 한국어 — 전 UI·데이터 대응.

### 🎨 라이트 / 다크 테마
- 무협 수묵 다크 테마 (기본).
- 화지조 라이트 테마 — S Tier 애니메이션 매끄럽게 + 대비 최적화 완료.

## Tier 판정 기준

장비 최적화 제안을 모두 적용했을 때의 최대 점수를 100% 로 하여、 현재의 무격지수 비율로 판정:

| Tier | 비율 |
|---|---|
| **SS** | 최대의 95% 이상 |
| **S** | 90% 이상 |
| **A** | 80% 이상 |
| **B** | 65% 이상 |
| **C** | 미만 |

## 계산에 반영되는 효과

- 장비 기본 스테이터스 (외공 공격력 / 무술 속성별 공격력 등)
- 조율 / 정음 옵션 (조율 1〜5、 정음 1〜5)
- 무술 재능 효과 (치명타율 상한 +Δ、 무술 속성별 공격 / 관통 / 속성 데미지 강화 등)
- 심법 Tier 효과 (T2/T5 = 표시 반영、 T0/T1/T3/T4/T6 = 숨김 가산)
- 장비 두 점 한 식 효과 (2개 장비로 발동하는 가산계)
- 장비 네 점 한 식 효과 (4개 장비로 +100 고정 보너스、 각 장비에 균등 배분)
- 기초값 (체/력/방/속/회) → 파생 (기본 스테이터스·판정 확률)

---

## 계산에 반영되지 않는 효과

- 장비 네 점 한 식 효과의 조건부 효과 (HP / 진기 / 받아넘김 / 강타 트리거 등) — 일률 +100 고정으로 대체
- 관음 (게임 내 스테이터스 비영향이 판명、 스테이터스 화면에 반영되지 않으므로 계산 대상 외)
- PvP 전용 정음 (정음 6 슬롯) — 표시만、 계산 기여 없음

## 사용법

1. **도구 열기** → https://wwm-metrics.pages.dev
2. 우상단 **「IMPORT」** 버튼 → 공식 데이터 도구 (확장) 에서 데이터 임포트
3. 장비 카드 / 심법 카드에 자동 반영
4. 장비 카드 클릭 → **장비 비교 modal** 에서 무술·장비 한 식·조율/정음 변경 → Δ Score 프리뷰
5. 격석 패널에서 기댓값 랭킹 / 효율 분석 / 최적화 확인
