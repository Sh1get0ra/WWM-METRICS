## 概要

**風燕伝 (Where Winds Meet)** 向けの装備強度・最適化分析ツールです。
ゲーム公式データツール (拡張) からインポートした装備情報をもとに、**武格指数 (Martial Index)** を中心に装備・調律/定音オプション・武術・心法・装備一式効果を統合してダメージ期待値を算出します。

> v2.0.0 で 旧版の手動ステータス入力 UI を廃止し、 装備データインポート駆動の新 UI に全面移行しました。

## 主な機能

### 📥 データインポート (完全自動)
- ゲーム公式データツール (拡張) から **ワンクリックでロールデータ取込** → 装備10点 / 調律・定音オプション / 心法 (4枠 + 武庫) / 武術 / キャラクター情報 を一括反映
- 手入力ゼロで武格指数を瞬時表示

### 🏯 武格指数 (Martial Index)
- 装備・調律/定音オプション・武術・心法・装備一式効果を全て込みで、 全プレイヤー共通の固定係数でダメージ期待値を算出した指標
- 大世界等級や敵側パラメータの個人差を排除 → 装備の絶対強度を比較可能

### ⚔️ 装備カード / 心法カード
- 装備カード: 武術属性別 (鋼鳴 / 砕岩 / 糸操 / 瞬嵐 / 無相) 配色 + セット効果名 + 武術名 + スコア
- 心法カード: Tier (T1〜T5) 朱印 chip + 心法名 + スコア

### 🪶 装備比較 modal (案A 巻物UI)
- 新置 (仮置) 装備でステ変動 → Δ Score 即時プレビュー
- MAX% rank別配色 (gold / purple / blue)、 **100% 到達時 虹色アニメ**
- PvP 専用定音 (定音6) 対応、 装備Lv上限はキャラクター Lv に制限

### 🧘 心法 modal
- 装備比較と統一 UI、 Tier 効果リスト (T0〜T6) + 武術属性専用 effects 反映

### 📊 格析パネル (タブ式)
- **期待値ランキング**: 調律/定音オプション +N の Δ Score 一覧
- **ステータス効率分析**: ステータス +1 あたりの ΔScore
- **装備最適化**: 全装備の Greedy 全体最適化提案
- **履歴**: 過去のインポート/Δ 一覧

### 💾 プリセット保存
- 複数スロット 保存/読込/削除
- importSnap + state + 仮想装備 + 基準値 (baseline) 同梱

### 📡 OBS Share / 配信用 URL
- 表示専用 URL (`?view=sidebar`) で sidebar のみ OBS overlay 化
- カラーカスタマイズ (背景/不透明度/文字色 等 6項目)

### 🌐 多言語 (4言語)
- 日本語 / English / 中文 (簡体) / 한국어 — 全 UI・データ対応

### 🎨 ライト / ダークテーマ
- 武侠水墨ダークテーマ (デフォルト)
- 和紙調ライトテーマ — Tier アニメ滑らか化 + コントラスト最適化済

## Tier 判定基準

装備最適化提案を全て適用した時の最大スコアを 100% として、 現在の武格指数の比率で判定:

| Tier | 比率 |
|---|---|
| **SS** | 最大の 95% 以上 |
| **S** | 90% 以上 |
| **A** | 80% 以上 |
| **B** | 65% 以上 |
| **C** | 未満 |

---

## 計算に反映される効果

- 装備 基本ステータス (外功攻撃力 / 武術属性別 攻撃力 等)
- 調律 / 定音オプション (調律 1〜5、 定音 1〜5)
- 武術才能効果 (会心率上限 +Δ、 武術属性別 攻撃 / 貫通 / 属性ダメ 強化 等)
- 心法 Tier 効果 (T2/T5 = 表示反映、 T0/T1/T3/T4/T6 = 裏加算)
- 装備二点一式効果 (2点装備で発動する加算系)
- 装備四点一式効果 (4点装備で +100 固定ボーナス、 各装備に均等配賦)
- 基礎値 (体/力/防/速/会) → 派生 (基本ステータス・判定確率)

---

## 計算に反映されない効果

- 装備四点一式効果の条件付効果 (気血/真気/受流/重撃 トリガー等) — 一律 +100 固定で代替
- 観音 (ゲーム内ステータス非影響と判明、 ステ画面に反映されないため計算対象外)
- PvP 専用定音 (定音6 枠) — 表示のみ、 計算寄与なし

## 使い方

1. **ツールを開く** → https://wwm-metrics.pages.dev
2. 右上 **「IMPORT」** ボタン → 公式データツール (拡張) からデータ取込
3. 装備カード / 心法カードに自動反映
4. 装備カードクリック → **装備比較 modal** で 武術・セット・調律/定音 変更 → Δ Score プレビュー
5. 格析パネルで 期待値ランキング / 効率分析 / 最適化 を確認
