## 概要

**風燕伝（Where Winds Meet）** 向けのリアルタイム・ダメージ計算ツールです。  
キャラクターのステータスや敵のパラメータを入力するだけで、期待ダメージ・各種判定確率・ステータス効率を即座に計算・表示します。

---

## 主な機能

### ⚔️ ダメージ期待値の計算
- **物理・属性ダメージの内訳**を割合付きで表示
- 外功攻撃・内功攻撃・各種ボーナス係数に対応
- 防御力・属性耐性・被ダメ減少を考慮した正確な計算

### 🎯 判定確率の分析
- 会心 / 会意 / かすり / 通常 の4種の確率をリアルタイム計算
- ドーナツチャートで確率分布を視覚的に表示
- 命中率・判定耐性による補正も自動反映

### 📊 ステータス効率分析
- 各ステータス（敏捷・勁力・腕力など）の投資対効果を比較
- 基礎ダメージとの差分・1あたりの上昇値を一覧表示
- **FIRE / ★★★ / GOOD** バッジで優先度を直感的に把握

### 💾 プリセット保存
- 3スロットのプリセット保存機能（ローカルストレージ）
- ビルドの比較・切り替えが瞬時に可能

### 🌐 多言語対応
- 日本語 / English / 中文 / 한국어 に対応

### 📱 レスポンシブデザイン
- PC・スマートフォン両対応
- 期待値パネルの折りたたみ表示で、スマホでも快適に操作可能

### 🎨 ライト / ダークテーマ
- 墨絵調ダークテーマ（デフォルト）
- 和紙調ライトテーマ

## 対応ステータス一覧

| カテゴリ | 入力項目 |
|---|---|
| 攻撃 | 物理攻撃（最小/最大）・内功攻撃（主/副） |
| スキル係数 | 外功係数・内功係数・付加ダメージ |
| 強化 | 全武技・特定武技・武器・対Boss・属性攻撃 |
| 判定 | 命中・会心・会意・追加会心/会意 |
| 会心/会意倍率 | 会心倍率・会意倍率 |
| 貫通 | 物理貫通・属性貫通 |
| 被ダメ軽減 | 被ダメ軽減①② |
| 敵パラメータ | 敵防御・判定耐性・物理/属性耐性（プリセットあり） |

## 使い方

1. **ツールを開く** → https://sh1get0ra.github.io/WWM-DMGCALC/
2. キャラクターのステータスをカードに入力
3. 敵レベル（またはカスタム値）を選択
4. **期待値・確率・効率**がリアルタイムで更新される
