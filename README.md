## 概要

**風燕伝 (Where Winds Meet)** 装備強度の比較・最適化ツールです。
風燕伝公式・データツールからインポートしたキャラクターステータスと、風燕伝ダメージ計算式をベースに **武格指数** を算出します。
最適な装備の提案、 装備一つ一つの比較など装備改善方針を提示します。

---

## 主な機能

### 📥 データインポート機能
- 風燕伝公式・データツールから **ワンクリックでキャラクター情報取込** → 調律・定音などを含む装備情報 / 装備中の心法 などのキャラクター情報 を一括反映
- 手入力ゼロで武格指数を瞬時表示

### 🏯 武格指数
- キャラクターステータスを元に、 全プレイヤー共通の固定スキル係数でダメージ期待値を算出した指標
- 武術のスキル倍率差などを固定化しているため、 装備の絶対強度を比較可能

### 🪶 装備比較
- インポートデータの装備と、新しい装備で 武格指数の変動をリアルタイムプレビュー
- 新しい装備については便利なOCR取込機能(全言語対応)実装

### 🧘 心法比較
- Tier別効果リスト + 武術属性専用効果 反映
- 現段階ではT2/T5以外の効果について、正確にスコア化するのは難しいため、グローバルダメージ係数を乗算

### 📊 格析パネル
- **期待値ランキング**: 現在装備を元に調律/定音の期待値ランキングが閲覧可能
- **装備最適化**: 現在装備からの最適な装備の提案/適用機能。

### 📈 武格履歴
- 過去のインポートと武格指数の推移をグラフ表示
- 複数キャラクター対応、 期間切替 (全期間 / 30日 / 7日)

### 💾 プリセット保存
- 複数スロット 保存/読込/削除
- 試行中の装備構成 + 基準値 を保存

### 📡 OBS Share
- カラーカスタマイズ可能なステータス情報をOBSで配信できるオーバーレイURLを発行。

### 🌐 多言語対応UI
- 日本語 / English / 中文 (簡体) / 한국어 / Tiếng Việt — 全 UI・データ対応
- ※表記がおかしいなどございましたらご連絡お願いいたします。

## 武格指数 Tier 判定基準

インポート時に適用された装備で算出された武格指数と最適化装備基準の武格指数の比率で判定。
武格指数 Tier はインポート時に自動判定されるため、 再インポートするまで変動しない。

| Tier | 比率 |
|---|---|
| **SS** | 最大の 95% 以上 |
| **S** | 90% 以上 |
| **A** | 80% 以上 |
| **B** | 65% 以上 |
| **C** | 未満 |

## 計算に反映される効果

- 装備 基本ステータス (外功攻撃力 / 武術属性別 攻撃力 等)
- 調律 / 定音効果
- 武術才能効果 (会心率上限 +Δ、 武術属性別 攻撃 / 貫通 / 属性ダメ 強化 等)
- 心法 Tier 効果 (T2/T5 = 表示反映、 T0/T1/T3/T4/T6 = 裏加算)
- 装備二点一式効果 (2点装備で発動する加算系)
- 装備四点一式効果 (4点装備で +100 固定ボーナス、 各装備に均等配賦)
- 基礎値 (体/力/防/速/会) → 派生 (基本ステータス・判定確率)
- キャラクター基本ステータス (才能 / 五音太平楽 / 観音 等)

---

## 計算に反映されない効果

- PvP 専用定音 (定音6 枠) — 表示のみ、 計算寄与なし

---

## 実機ステータスとの誤差について

武格指数は キャラクター基本ステータス (才能 / 五音太平楽 / 観音 等) を **全プレイヤー共通の MAX想定** で算出します。
実機キャラがまだ振り切れていない場合、 ツール上の数値が実機より高く出ることがあります。
これは絶対強度比較を成立させるための仕様です。
