## Anti-Patterns Verdict

**AI 製に見えるか: 見えない。** 朱墨軍議の identity が全面に通り、 SaaS-cream / eyebrow / hero-metric template / カード羅列の気配ゼロ。 印章・縦書き rail・Cinzel 数字は訓練データ反射の対極。

**Deterministic scan (CLI 4 件)**: 文脈上ほぼ全て false positive — ①FAB avatar img src 無し (JS が import 後に注入、 それまで [hidden]) ②single-font "cinzel" (Noto/Rajdhani は system stack で検出外) ③em-dash 7 (ja ダーシ用法) ④numbered markers 01/02/03 (= preset slot 番号 = 実シーケンス、 scaffold でない)。

**Browser overlay (53 logs)**: 大半 = identity 由来の FP (dark-glow ×12 = 階級の光 / clipped-overflow ×20 = 装飾 clip 意図)。 **実所見**: low-contrast 4 件 (下記 P2)、 tiny-text 11px body ×2 + 0.30em tracking、 gradient-text 1 件 (勁率 donut center label)、 side-stripe 3px 1 件 (affix rank row)、 bounce-easing ×2 (cubic-bezier(0.2,1.4,0.5,1))。

## Overall Impression

世界観の実装力は本物。 数字が主役の階層も完璧に機能している。 最大の弱点は美学でなく**縦 responsive**: 1440×900 / 1366×768 (最多級のラップトップ) で中核機能の最適化提案 panel が 42px に圧潰し 13 行が無 scroll で不可視になる。 ここを直せば 34-35 点台が見える。

## Priority Issues

- **[P1] anlz panel が短い viewport で圧潰**: 900px 高で `#wwmOptimization .wwm-analysis-card` = 42px (overflow hidden、 scroll 不可、 13 行不可視)。 1080p でも可視 ~3.5 行。 hero 折りたたみも実質効かない (collapsed で cardH 42 のまま)。 **Why**: 中核機能が最多級ラップトップで「存在しない」 ように見える。 mobile overlay の方が情報量が多い逆転。 **Fix**: `.wwm-anlz` 系に min-height (例 320px) を与え `.container` の overflow-y:auto に流す、 または card 内部 scroll。 **Command**: /impeccable adapt
- **[P2] topbar 状態表現が弱い**: 言語 active (TODO #24 known、 dark/light 両方) + theme toggle glyph が "C" に見える。 **Why**: 現在状態の発見性 = Heuristic 1/4 の失点源。 **Fix**: active に下線/枠/濃度差、 toggle は ☀/☾ icon 明示。 **Command**: /impeccable polish
- **[P2] contrast cluster (detector 実測)**: paper-mute #7a7060 on bg-2 #16110f = 3.8:1 (小サイズ本文で AA 未達 — bg-0 比では達するが浮上面で陥落) / gold on vermilion 2.2:1 (state/pseudo 依存、 mobile 格析 btn 疑い) / #ede4d0・#d4af37 on #f0d28a 1.2-1.4:1 (state 依存)。 11px body + 0.30em tracking も連座。 **Why**: 高密度 HUD は小文字依存 = contrast が生命線。 **Fix**: 浮上面用 paper-mute 段階の token 追加 + 該当 state 特定。 **Command**: /impeccable audit
- **[P2] mobile touch target < 44px**: preset 行の 💾📁× icon 群 ~30px、 topbar 小 btn (TODO #14 残)。 **Command**: /impeccable adapt
- **[P3] 小粒 UX 群**: anlz empty state = 無案内の空箱 (「IMPORT で表示」 一行欲しい) / 弱点 badge が empty で「0」 表示・populated で非表示の非対称 / hero collapse の存在意義 (折りたたみで何も縮まない) / 勁率 gradient-text・rank row side-stripe 3px・bounce easing ×2 の禁則触れ (意匠意図はあるが規約上は要判断)。 **Command**: /impeccable onboard + polish
