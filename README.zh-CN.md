## 概述

**燕云十六声 (Where Winds Meet)** 装备强度·最优化分析工具。
基于游戏官方数据工具 (扩展) 导入的角色数据、 以 **武格指数 (Martial Index)** 为核心、 整合装备 / 调律·定音词条 / 武学 / 心法 / 装备一式效果 计算伤害期望值。

> v2.0.0 已废除旧版手动输入界面、 全面迁移至装备数据驱动的新 UI。

---

## 主要功能

### 📥 数据导入 (全自动)
- 通过游戏官方数据工具 (扩展)、 **一键导入角色数据** → 装备10件 / 调律·定音词条 / 心法 (4槽 + 武库) / 武学 / 角色信息 全部反映。
- 零手动输入、 武格指数即时显示。

### 🏯 武格指数 (Martial Index)
- 装备·调律/定音词条·武学·心法·装备一式效果 全部纳入、 以全玩家通用的 **固定技能系数** 计算伤害期望值。
- 排除大世界等级和敌方参数的个人差异 → 可比较装备的绝对强度。

### ⚔️ 装备卡 / 心法卡
- 装备卡: 武学属性别 (钢鸣 / 碎岩 / 丝操 / 瞬岚 / 无相) 配色 + 装备一式效果名 + 武学名 + 分数。
- 心法卡: Tier (T1〜T5) 朱印 chip + 心法名 + 分数。

### 🪶 装备对照 modal (卷轴UI)
- 新置 (试置) 装备引起的属性变化 → Δ Score 即时预览。
- MAX% rank 别配色 (金 / 紫 / 蓝)、 **达到100%时彩虹色相旋转动画**。
- PvP专用定音 (定音6) 对应、 新置装备等级上限为角色等级。

### 🧘 心法 modal
- 与装备对照统一 UI、 Tier 效果列表 (T0〜T6) + 武学属性专用 effects 反映。

### 📊 格析面板 (标签式)
- **期望值排行**: 调律/定音词条 +N 的 Δ Score 列表。
- **状态效率分析**: 每 +1 状态点的 ΔScore。
- **装备最优化**: 全装备 Greedy 整体最优化建议。
- **历史记录**: 过去导入 / Δ 列表。

### 💾 预设保存
- 多槽位 保存 / 读取 / 删除。
- importSnap + state + 虚拟装备 + 基准值 (baseline) 同捆。

### 📡 OBS Share / 直播URL
- 显示专用 URL (`?view=sidebar`) 用于 OBS overlay (仅 sidebar)。
- 颜色自定义 (背景 / 不透明度 / 文字色 等 6项)。

### 🌐 多语言 (4语言)
- 日本語 / English / 简体中文 / 한국어 — 全 UI·数据 对应。

### 🎨 浅色 / 深色主题
- 武侠水墨深色主题 (默认)。
- 和纸调浅色主题 — S Tier 动画平滑化 + 对比度优化完成。

## Tier 判定基准

以 **装备最优化建议全部应用时的最大分数** 为100%、 按当前武格指数比率判定:

| Tier | 比率 |
|---|---|
| **SS** | 最大的 95% 以上 |
| **S** | 90% 以上 |
| **A** | 80% 以上 |
| **B** | 65% 以上 |
| **C** | 未满 |

## 计算反映的效果

- 装备 基础状态 (外功攻击力 / 武学属性别 攻击力 等)
- 调律 / 定音词条 (调律 1〜5、 定音 1〜5)
- 武学才能效果 (会心率上限 +Δ、 武学属性别 攻击 / 穿透 / 属性伤害 强化 等)
- 心法 Tier 效果 (T2/T5 = 显示反映、 T0/T1/T3/T4/T6 = 隐藏加算)
- 装备二点一式效果 (2件装备发动的加算系)
- 装备四点一式效果 (4件装备 +100 固定奖励、 各装备均匀分配)
- 基础值 (体/力/防/速/会) → 派生 (基础状态·判定概率)

---

## 计算不反映的效果

- 装备四点一式效果的条件付效果 (气血/真气/受流/重击 触发等) — 一律 +100 固定代替
- 观音 (已确认不影响游戏内状态画面、 因此排除)
- PvP 专用定音 (定音6 槽) — 仅显示、 无计算贡献

## 使用方法

1. **打开工具** → https://wwm-metrics.pages.dev
2. 右上 **「IMPORT」** 按钮 → 从官方数据工具 (扩展) 导入数据
3. 装备卡 / 心法卡 自动反映
4. 点击装备卡 → **装备对照 modal** → 变更 武学·装备一式·调律/定音 → Δ Score 预览
5. 在格析面板查看 期望值排行 / 效率分析 / 最优化
