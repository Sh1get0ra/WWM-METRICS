## 概述

**《燕雲十六聲》(Where Winds Meet)** 装备强度比较·最优化工具。
基于从《燕雲十六聲》官方數據小工具导入的角色属性，依据《燕雲十六聲》伤害公式计算 **武格指数**。
提供最优装备建议、装备逐件比较等装备改进方向。

---

## 主要功能

### 📥 数据导入
- 通过《燕雲十六聲》官方數據小工具 **一键导入角色信息** → 包含调律·定音的装备信息 / 装备中的心法 等角色信息 一并反映。
- 零手动输入、 武格指数即时显示。

### 🏯 武格指数
- 基于角色属性、 以全玩家通用的固定技能系数计算伤害期望值的指标。
- 固定了武学技能倍率等差异、 因此可比较装备的绝对强度。

### 🪶 装备对照
- 实时预览导入装备与新装备之间的武格指数变动。
- 新装备录入支持便利的 OCR 取入功能 (全语言对应)。

### 🧘 心法对照
- Tier 别效果列表 + 武学属性专用效果 反映。
- 现阶段除 T2/T5 以外的效果难以准确分数化、 因此乘以全局伤害系数。

### 📊 格析面板
- **期望值排行**: 基于当前装备查看调律/定音期望值排行。
- **装备最优化**: 基于当前装备的最优装备建议/应用功能。

### 📈 武格历史
- 过去导入与武格指数变迁 以图表显示。
- 支持多角色、 期间切换 (全期间 / 30日 / 7日)。

### 💾 预设保存
- 多槽位 保存 / 读取 / 删除。
- 保存试行中的装备配置 + 基准值。

### 📡 OBS Share
- 发行可自定义颜色、 用于 OBS 直播状态信息的 overlay URL。

### 🌐 多语言对应 UI
- 日本語 / English / 简体中文 / 한국어 / Tiếng Việt — 全 UI·数据 对应。
- ※ 若发现表记有误、 请联系我们。

## 武格指数 Tier 判定基准

按导入时所应用装备算出的武格指数与最优化装备基准的武格指数的比率判定。
武格指数 Tier 在导入时自动判定、 再次导入前不会变动。

| Tier | 比率 |
|---|---|
| **SS** | 最大的 95% 以上 |
| **S** | 90% 以上 |
| **A** | 80% 以上 |
| **B** | 65% 以上 |
| **C** | 未满 |

## 计算中反映的效果

- 装备基础属性 (外功攻击力 / 武学属性别 攻击力 等)
- 调律 / 定音效果
- 武学才能效果 (会心率上限 +Δ、 武学属性别 攻击 / 穿透 / 属性伤害 强化 等)
- 心法 Tier 效果 (T2/T5 = 显示反映、 T0/T1/T3/T4/T6 = 隐藏加算)
- 二件套效果 (2件装备发动的加算系)
- 四件套效果 (4件装备 +100 固定加成、 各装备均分)
- 基础值 (体/力/防/速/会) → 派生 (基础属性·判定概率)
- 角色基础属性 (天赋 / 五音启太平 / 观音 等)

---

## 计算中未反映的效果

- PvP 专用定音 (定音 6 槽) — 仅显示、 不参与计算

---

## 关于与游戏内数值的差异

武格指数将角色基础属性 (天赋 / 五音启太平 / 观音 等) 视为 **所有玩家通用的最大值** 进行计算。
若角色尚未达到最大值、 工具数值可能高于游戏内实际数值。
此为实现装备绝对强度比较的设计。
